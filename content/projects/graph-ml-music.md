## Overview
This project explores how graph deep learning can model relationships between songs, artists, harmonic patterns, and stylistic tendencies.

## Motivation
I wanted to combine my interests in machine learning and music into a project that is both technically interesting and creatively meaningful.

## Methods
- heterogeneous graph construction
- node and edge feature design
- graph neural networks for prediction and embedding

## Tools
Python, PyTorch Geometric, NetworkX, pandas

## Outcome
The goal is to better understand stylistic similarity, genre structure, and musical tendencies using graph representations.

## Next Steps
- expand dataset coverage
- improve feature engineering
- test embedding quality with downstream tasks
